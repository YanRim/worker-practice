# Worker Practice

Практическая работа по программированию на C++.

## Описание

Программа предназначена для учета работников ООО "ПК Вентстрой Плюс".

Реализован класс WORKER, содержащий следующие поля:

- фамилия и инициалы работника;
- должность;
- заработная плата;
- год поступления на работу.

## Возможности программы

- ввод данных работников с клавиатуры;
- хранение работников в стандартном списке list;
- расчет стажа работы;
- вывод работников, чей стаж превышает значение, введенное пользователем.

## Файлы проекта

- Worker.h — описание класса WORKER;
- Worker.cpp — реализация методов класса;
- main.cpp — основная программа.

## Язык программирования

C++.
